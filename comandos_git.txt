git init. -> cria um repo git na pasta atual
git status -> mostra o status do trabalho
git add nome_arquivo -> adiciona arquivo para ser commitado
git commit -m -> faz commit do que estava para ser commitado
git log -> mostra todo o histórico de commit


### Fluxo de Trabalho
1.Criar o diretório (pasta) do projeto
2.Iniciar o git dentro do diretório criado (git init .)
3.Edita ou Cria arquivos;
4.Git Status
5.Git Add <nome do arquivo>;
6.Git Status;
7.Git commit -m “mensagem”;
8.Git Status
9.Volta para o passo 3;
